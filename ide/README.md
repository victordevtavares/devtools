# Visual Code Studio

1. Para iniciar o VSC pelo terminal execute o comando code ou code seguido do nome da pasta onde deseja trabalhar ou arquivo especifico exemplo code index.html .

1. No VSC o terminal tambem esta disponivel nele os comandos serao os mesmos de sua maquina voce podera abrir o terminal clicando em terminal e criando um novo terminal.

1. No lado esquerdo do VSC voce pode ver suas extensoes instaladas caso queira instalar novas extensoes clique nos 4 quadradinhos e digite na busca a extensao que deseja entao clique em instalar exemplo um tema que gosto e o palenight.
